<h1>Javascript Terminal Project</h1>

<p>
This repository contains multiple <strong>terminal-based JavaScript applications</strong> built using pure Node.js.  
All projects are designed to run directly in your terminal and help beginners practice JavaScript, user input handling, and CLI interactions.
</p>

<h2 id="overview">Overview</h2>
<p>This repository currently includes the following terminal applications:</p>

<ul>
    <li><strong>Random Phrase Generator</strong>: Displays a random phrase on each execution.</li>
    <li><strong>Rock Paper Scissors Game</strong>: Classic CLI game where you play against the computer.</li>
    <li><strong>Password Generator</strong>: Generates secure random passwords with different lengths.</li>
    <li><strong>To-Do List App</strong>: Add, view, delete, and manage tasks directly in your terminal.</li>
</ul>

<h2 id="features">Features</h2>
<ul>
    <li>Fully interactive terminal applications</li>
    <li>Beginner-friendly JavaScript implementations</li>
    <li>No external libraries required</li>
    <li>Simple file structure</li>
    <li>Reusable logic across multiple small projects</li>
</ul>

<h2 id="getting-started">Getting Started</h2>

<h3>Prerequisites</h3>
<ul>
    <li><strong>Node.js</strong> installed on your machine</li>
    <li>Basic familiarity with using a terminal</li>
</ul>

<h3>Installation</h3>
<pre><code>git clone https://github.com/ghanashyambudhathoki01/Javascript-Terminal-Project.git
cd Javascript-Terminal-Project
</code></pre>

<h2 id="individual-projects">Individual Project Usage</h2>

<h3>1. Random Phrase Generator</h3>
<pre><code>cd "Random Pharse Generator"
node index.js
</code></pre>

<p><strong>What it does:</strong> Displays a new random phrase each time you run the script.</p>

<hr>

<h3>2. Rock Paper Scissors Game</h3>
<pre><code>cd "Rock Paper Scissor Game"
node index.js
</code></pre>

<p><strong>Gameplay Logic:</strong></p>
<ul>
    <li>User chooses Rock, Paper, or Scissors</li>
    <li>Computer randomly selects a move</li>
    <li>Winner is decided based on classic rules</li>
    <li>Option to replay</li>
</ul>

<hr>

<h3>3. Password Generator</h3>
<pre><code>cd "Password Generator"
node index.js
</code></pre>

<p><strong>Features:</strong></p>
<ul>
    <li>Generates secure random passwords</li>
    <li>Supports different password lengths</li>
    <li>Fast, simple, and useful for learning algorithm basics</li>
</ul>

<hr>

<h3>4. To-Do List (Terminal Application)</h3>
<pre><code>cd "To-Do-List Js"
node index.js
</code></pre>

<p><strong>Features:</strong></p>
<ul>
    <li>Add new tasks</li>
    <li>View all tasks</li>
    <li>Delete tasks</li>
    <li>Lightweight and easy to expand</li>
</ul>
